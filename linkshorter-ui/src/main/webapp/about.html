Скорочення URL — можливість, що використовується в Інтернеті для того, щоб мати додаткову, альтернативну коротку URL-адресу для доступу до веб-сторінки. Кожній URL-адресі присвоюється ключ, який додається після http://l.co.ua/. Наприклад, http://l.co.ua/m3q2xt має ключ m3q2xt.

Short URL - additional, alternative short URL-address for accessing the web page.             Each URL-address assigned to the key that is added after http://l.co.ua/. For example, http://l.co.ua/m3q2xt a key m3q2xt.

Сокращение URL — возможность, что используется в Интернете для того, чтобы иметь альтернативный краткий URL-адрес для доступа к веб-странице. Каждому URL-адресу присваивается ключ, который прибавляется после http://l.co.ua/. Например, http://l.co.ua/m3q2xt имеет ключ m3q2xt.
