REST v. 0.2.1:

post

  • http://l.co.ua/rest/postUrl
    post long url and get short id
  • http://l.co.ua/generateShort
    post long url and get short id (DEPRECATED)

get

  • http://l.co.ua/[id]
    get redirect html to long url with id
  • http://l.co.ua/rest/statistic/[id]
    get statistic for id. Statistic is array in JSON format, each field contains information about: Date, Country and OS

limits

  • max 3 per second;
  • max 20 per minute;
  • max 30 per hour;
  • max 100 per day.
